关于hugo的一切，你都可以查看其官网 来了解 ，建议是跟着它的 quick-start一步一步来操作

安装

hugo通过Github发布，这里我推荐你 下载 extended版本，而且最好是.tar.gz或.zip版本的，并且设置环境变量，将目录添加到你的Path变量中，以便命令行能够找到hugo

创建博客生成器

1

hugo new site xxxxx.github.io-creator

将目录用git管理起来

1
2

cd xxxxx.github.io-creator 
git init

选择一个主题

hugo有很多主题你可以根据喜好自己选一个，推荐选择配置简单的,我用的 meme

1

git submodule add https://github.com/reuixiy/hugo-theme-meme.git themes/meme

用Markdown写文章

1

hugo new posts/my-first-post.md
  • 然后打开content/posts/my-first-post.md写就OK了

预览效果

1

hugo server -D

发布博客

1

hugo -D
  • 这会在其目录生成public文件夹，记得用git管理起来

上传到Github

  1. 在Github上创建一个xxxx.github.io的仓库（必须是Github page）
  2. 然后将你的public中的内容push上去吧
  3. 最后你访问xxxx.github.io应该就能看到效果了，祝你好运!
